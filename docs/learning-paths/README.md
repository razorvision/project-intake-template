## üìö Learning Path Details

### 1. New to Claude Code

**Lessons:**
1. [Setup and Configuration](new-to-claude-code/01-setup.md) (20 min)
   - Install and configure Claude Code
   - Set up permissions and CLAUDE.md
   - Configure MCP servers

2. [Your First Task](new-to-claude-code/02-first-task.md) (30 min)
   - Navigate a codebase
   - Make your first edit
   - Run tests and commit

3. [Using MCP Effectively](new-to-claude-code/03-using-mcp.md) (30 min)
   - When to use which MCP server
   - Browser automation with Playwright
   - Documentation lookup with Context7

4. [Master the Workflows](new-to-claude-code/04-workflows.md) (30 min)
   - Explore ‚Üí Plan ‚Üí Code ‚Üí Commit
   - Test-driven development
   - Visual iteration

5. [Best Practices](new-to-claude-code/05-best-practices.md) (20 min)
   - Effective prompting
   - Security considerations
   - Team collaboration

**Prerequisites:** None
**What you'll build:** A complete feature from scratch

---

### 2. Testing Fundamentals

**Lessons:**
1. [Unit Testing Basics](testing-fundamentals/01-unit-testing.md) (45 min)
   - Writing testable code
   - Jest/Vitest basics
   - Mocking and assertions

2. [Integration Testing](testing-fundamentals/02-integration-testing.md) (45 min)
   - Testing multiple components
   - Database testing
   - API testing

3. [End-to-End Testing](testing-fundamentals/03-e2e-testing.md) (60 min)
   - Playwright setup
   - Writing E2E tests
   - CI/CD integration

4. [Testing Strategy](testing-fundamentals/04-testing-strategy.md) (30 min)
   - Test pyramid
   - Coverage goals
   - Continuous improvement

**Prerequisites:** Basic JavaScript/TypeScript
**What you'll build:** Complete test suite for a feature

---

### 3. Authentication Deep Dive

**Lessons:**
1. [Auth Basics](authentication-deep-dive/01-auth-basics.md) (45 min)
   - Authentication vs. authorization
   - Session vs. JWT tokens
   - OAuth and social providers

2. [Session Management](authentication-deep-dive/02-session-management.md) (45 min)
   - Session storage strategies
   - Token refresh patterns
   - Logout and session cleanup

3. [Security Patterns](authentication-deep-dive/03-security-patterns.md) (60 min)
   - Password hashing
   - CSRF protection
   - Rate limiting
   - Security headers

4. [Common Pitfalls](authentication-deep-dive/04-common-pitfalls.md) (30 min)
   - Debugging auth issues
   - Session timeout handling
   - Multi-device auth
   - Production considerations

**Prerequisites:** Basic web development knowledge
**What you'll build:** Production-ready auth system

---

### 4. API Design

**Lessons:**
1. [REST Fundamentals](api-design/01-rest-fundamentals.md) (45 min)
   - HTTP methods and status codes
   - Resource naming
   - RESTful patterns

2. [Error Handling](api-design/02-error-handling.md) (45 min)
   - Error response formats
   - Validation errors
   - Error logging and monitoring

3. [Versioning and Evolution](api-design/03-versioning.md) (45 min)
   - API versioning strategies
   - Breaking vs. non-breaking changes
   - Deprecation process

4. [Advanced Patterns](api-design/04-advanced-patterns.md) (45 min)
   - Pagination and filtering
   - Rate limiting
   - Webhooks
   - GraphQL considerations

**Prerequisites:** Backend development basics
**What you'll build:** Complete RESTful API

---

### 5. Scaling to Production

**Lessons:**

   - Profiling and benchmarking

### 2. Follow in Order
Lessons build on each other. Complete them sequentially.

### 3. Hands-On Practice
Each lesson includes exercises. Do them‚Äîdon't just read!
